## Dataset Description
**Wisconsin Diagnostic Breast Cancer (WDBC) Dataset**
- Samples: 569 (212 malignant, 357 benign)
- Features: 30 numeric predictors
  - Mean, standard error and worst values of:
    - Radius
    - Texture
    - Perimeter
    - Area
    - Smoothness
    - Compactness
    - Concavity
    - Symmetry
    - Fractal dimension
- Target: Binary diagnosis (0 = benign, 1 = malignant)
